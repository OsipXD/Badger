# Здесь должны быть данные о людях в таком формате:
#      [Фамилия Имя] (Отчество) : [Тип бейджа] : (Доп. текст) - (Личная цитата)
# [] - Обязательное, () - Необязательное
# Фамилия Имя Отчество: пишутся именно в таком порядке и с большей буквы
# Типы бейджей: школьник, сотрудник, гость, некто (можно использовать сокращения, главное чтобы начиналось на правильную букву)
# Доп. текст: тут можно написать должность человека
# Личная цитата: пишется после тире, цитата для этого ч-ка не рандомится
#
# Примеры правильных данных:
#      Ктулху Владыка Миров : некто - Вот проснусь и всех зохаваю. И никакой Чак Норрис не помешает.
#      Иванов Пётр : шк.
#      Ройтберг Михаил Абрамович : создатель : Директор ЗПШ
#
# Строки, начинающиеся с '#' считаются комментариями и не учитываются

Ктулху Владыка Миров : некто - Вот проснусь и всех зохаваю. И никакой Чак Норрис не помешает.
Иванов Пётр : шк.
Ройтберг Михаил Абрамович : создатель : Директор ЗПШ
